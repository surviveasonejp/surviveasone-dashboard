# Survive as One Japan

ホルムズ海峡封鎖時に日本のエネルギーがどう崩壊するかを可視化し、市民の生存判断を支援する戦術ダッシュボード。

**Live:** [surviveasonejp.org](https://surviveasonejp.org)

## Features

- **Survival Clock** -- 石油/LNG/電力の残存日数カウントダウン
- **Collapse Map** -- 全国10電力エリアの崩壊順マップ
- **Last Tanker Tracker** -- 日本向け最終タンカー到着予測
- **Food Chain Collapse** -- 商品カテゴリ別消失予測
- **Family Survival Meter** -- 家庭の生存可能日数を算出
- **備蓄ガイド** -- フェーズ別行動指針

## Tech Stack

React 19 + Vite 6 + TypeScript (strict) / Cloudflare Workers + D1 + KV + R2
